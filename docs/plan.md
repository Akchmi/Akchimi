# 게시판 관련 기능

## 공지사항 게시판

- 관리자/스텝만 작성 가능
- 댓글 기능 없음

## 자유 게시판

- 장비 공유
- 방송 세팅하는 법
- 노하우
- 댓글, 좋아요

- 악보
- 메트로놈
- 녹음, 녹화
- 즉석 다시보기
- 채팅
- 전자 악기 연결

## 연습실

- 악보
- 점수
- 거울 모드
- 연습 시간 기록
- 일주일 전에 너는

## 강의 신청 프로세스

강사검색(메인화면) → 강의신청(강사 상세정보 - 원하는시간대 선택) → 강사 수락 → 강의성사

## 강의 시작 프로세스

내강의(마이페이지) → 강사가 방생성 → 방생성시 학생한테 링크 전송 → 학생 접속 → 강의종료시 학생퇴실, 강사가 나가면 방 터지기

# 자유게시판
